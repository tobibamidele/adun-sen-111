Topic: Introduction to Web Development

Lecturer: Mr Bernard Ephraim

Level: 100

Summary:

The course began with a foundational look at web development, defined as the tasks associated with creating, building, and maintaining websites and web applications.

1. Types of development: The field is categorized into front-end (client-side), back-end (server-side), and full-stack (both).

2. Website Classification: Websites can be static (fixed content) or dynamic (interactive and database-driven and rely on server side programming languages).

3. Requirements: Web development necessitates specific hardware, such as Server (central computers serving responses) and Client (user devices with browsers), alongside software like text editors (Sublime Text, Notepad++), HTML, CSS and Javascript.

Topic: Basic HTML Document Structure

Lecturer: Mr Bernard Ephraim

Level: 100

Summary:

This week focused on HTML (HyperText Markup Language) document structure, including elements, tags, and basic syntax.

1. Structure: A standard HTML document uses the .html or .htm file extension. and follows a specific layout including <!DOCTYPE HTML> (indicating HTML5) and other tags for page content.

2. Tags and Syntax: The anatomy of opening anc closing tags noting attributes like class, id, and style.

Topic: Introduction to Cascading Style Sheets (CSS)

Lecturer: Mr Bernard Ephraim

Level: 100

Summary:

Week 3 introduced CSS a stylesheet language used to describe the presentation and outlook of HTML objects

1. Modes of CSS: There are 3 ways to apply styles.
i) Inline styling (directly in the HTML element)
ii) Internal styling Withing a style tag in the head section of the HTML document.
iii) External styling Linked to a seperated .css file.

Topic: Introduction to JavaScript

Lecturer: Mr Bernard Ephraim

Level: 100

Summary:

Week 6 introduced JavaScript, a programming language that enables interactive web pages and dynamic content.

1. Programming concepts: Concepts such as variables (declared by var), literals (strings or numbers), arrays (ordered collections), and objects (collections with keys) were introduced..

2. Logic and Control: The class covered mathematical operators (following BODMAS), comparators (==, !=, <, >) and logical operators (&&, ||, !).

3. Functions and Events: Functions are reusable blocks of code, and events are actions that occur in response to user interactions (e.g., clicks, keypresses).

4. Conditional Statements and Loops: These control the flow of code execution based on conditions (if, else if, else) and allow for repeated execution (for, while).

5. Outputting data: Methods like console.log() (for debugging) and document.write() (for writing to the webpage) were covered.
